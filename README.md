# Probable Umbrella

Anti flat-earth indoctrination.

### Dependencies:
* [Node](https://nodejs.org) ~> v9.2 (earlier will probably work too, but I haven't tested any others)
* [Yarn](https://yarnpkg.com)

#### Installing the deps:
```bash
yarn
```

#### Running the script:
```bash
yarn start
```

#### Running the tests:
```bash
yarn test
```

#### Running the tests with coverage reporting:
```bash
yarn coverage
```
